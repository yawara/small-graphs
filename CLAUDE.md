# CLAUDE.md

このリポジトリで作業する Claude / Codex / その他のエージェント向けの指示です。

## プロジェクトの目的

`small-graphs` は、degree-diameter problem に関係する小さい有限グラフの性質を、日本語の自然言語で整理するためのリポジトリです。

最終的には Lean による形式化を目指しますが、現段階では Markdown による数学的記述を主成果物とします。

## 作業時の原則

- 本文は日本語で書いてください。
- 数学的主張は、可能な限り定義、命題、証明スケッチ、補題候補に分けてください。
- Lean 化を意識して、曖昧な言葉を避けてください。
- 頂点番号、辺集合、写像、群の生成元は明示的に書いてください。
- 既存の Markdown を大きく書き換えるときは、数学的意味を変えていないか注意してください。
- 誤りを見つけた場合は、遠慮なく修正し、なぜ誤りかを短く説明してください。

## Markdown メモの形式

新しいグラフのメモを追加する場合は、`notes/` 以下に置いてください。

ファイル名は英小文字とハイフンを使い、例として次のようにします。

```text
notes/mobius-ladder-m8.md
notes/petersen-graph.md
notes/cube-q3.md
```

## Lean 化時の方針

Lean ファイルを追加する段階では、まず自然言語ノート内の命題に対応する小さな補題を作ってください。

特に、次の順に進めるのが望ましいです。

1. 頂点型の定義
2. 隣接関係の定義
3. 単純グラフとしての構成
4. 次数の証明
5. 直径の証明
6. 自己同型の構成
7. 自己同型群の位数または同型型の証明

まだ Lean 化していない自然言語メモを、勝手に完全証明済みとして扱わないでください。
