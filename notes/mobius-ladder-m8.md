# Möbius ladder M8 / Wagner graph

このノートでは、8頂点・3正則・直径2のグラフとして、Möbius ladder `M8` を記述する。

このグラフは Wagner graph とも呼ばれる。

## 定義

頂点集合を

```text
V = Z/8Z = {0,1,2,3,4,5,6,7}
```

とする。

辺は次の二種類からなる。

1. 8-cycle の辺

```text
i ~ i+1 mod 8
```

2. 対向頂点を結ぶ辺

```text
i ~ i+4 mod 8
```

ただし無向グラフなので、重複を避けて辺集合を書くと

```text
E = {
  01, 12, 23, 34, 45, 56, 67, 70,
  04, 15, 26, 37
}
```

である。

同値に、隣接関係は

```text
i ~ j  iff  j - i ≡ ±1 or 4 mod 8
```

で与えられる。

## 基本性質

このグラフは8頂点の単純無向グラフである。

各頂点 `i` の近傍は

```text
N(i) = {i-1, i+1, i+4}
```

である。ただし添字は全て mod 8 で読む。

したがって各頂点の次数は3である。よって `M8` は3正則グラフである。

## 直径

`M8` の直径は2である。

頂点推移性を使えば、頂点 `0` から全ての頂点までの距離を見れば十分である。

頂点 `0` の近傍は

```text
N(0) = {1,7,4}
```

である。

距離2で到達できる頂点は、例えば次の通りである。

```text
0 -> 1 -> 2
0 -> 7 -> 6
0 -> 1 -> 5
0 -> 7 -> 3
```

したがって、頂点 `0` から見た距離層は

```text
距離0: {0}
距離1: {1,7,4}
距離2: {2,3,5,6}
```

であり、全ての頂点が距離2以内に入る。

一方、このグラフは完全グラフではないので直径は1ではない。例えば `0` と `2` は隣接していない。

従って

```text
diam(M8) = 2
```

である。

## 3正則・8頂点・直径2の例としての意味

次数3、直径2の Moore bound は

```text
1 + 3 + 3(3-1) = 10
```

である。

`M8` はこの上限10には達しないが、8頂点で次数3かつ直径2を満たす具体例である。

注意点として、8頂点3正則グラフとしては立方体グラフ `Q3` も自然に現れるが、`Q3` の直径は3である。従って、8頂点3正則だからといって直径2とは限らないし、逆に8頂点3正則グラフが `Q3` に限られるわけでもない。

## 自己同型群

頂点を `Z/8Z` として、次の二つの写像を考える。

```text
r(i) = i + 1 mod 8
s(i) = -i mod 8
```

`r` は8-cycle の回転であり、`s` は反転である。どちらも隣接関係

```text
j - i ≡ ±1 or 4 mod 8
```

を保つため、自己同型である。

これらは関係式

```text
r^8 = 1
s^2 = 1
srs = r^{-1}
```

を満たす。

従って、少なくとも位数16の二面体群

```text
<r, s | r^8 = s^2 = 1, srs = r^{-1}>
```

が自己同型群に含まれる。

実際には、これが全ての自己同型である。

### 頂点安定化群による確認

グラフは `r` により頂点推移的である。よって、自己同型群の位数を求めるには、頂点 `0` の安定化群を調べればよい。

頂点 `0` の近傍は

```text
N(0) = {1,7,4}
```

である。

ここで辺 `0--4` は対向頂点を結ぶ chord であり、辺 `0--1` や `0--7` は外周の cycle 辺である。

この違いはグラフ構造から検出できる。例えば、辺 `0--4` は二つの4-cycle に含まれる。

```text
0-1-5-4-0
0-7-3-4-0
```

一方、辺 `0--1` は一つの4-cycle

```text
0-1-5-4-0
```

に含まれる。同様に、辺 `0--7` も一つの4-cycle に含まれる。

したがって、`0` を固定する自己同型は、近傍 `{1,7,4}` の中で `4` を固定しなければならない。

残る `1` と `7` は固定されるか入れ替わるかの二通りである。実際、反転

```text
s(i) = -i
```

は `0` と `4` を固定し、`1` と `7` を入れ替える。

従って、頂点 `0` の安定化群は位数2である。

軌道安定化定理より

```text
|Aut(M8)| = 8 * 2 = 16
```

である。

結論として

```text
Aut(M8) ≅ C8 ⋊ C2
```

であり、これは正八角形の対称群、すなわち位数16の二面体群である。

記法の曖昧さを避けるなら

```text
Aut(M8) ≅ <r, s | r^8 = s^2 = 1, srs = r^{-1}>
```

と書くのが安全である。

## 辺推移性について

このグラフは頂点推移的であるが、辺推移的ではない。

外周の辺と対向 chord の辺は、グラフ内部で区別できる。例えば、含まれる4-cycle の本数によって区別できる。

したがって、自己同型は外周辺を chord に移すことはできない。

## Lean 化のための補題候補

### 頂点と隣接関係

- 頂点型を `ZMod 8` または `Fin 8` で定義する。
- 隣接関係を `j - i = 1 or j - i = -1 or j - i = 4` として定義する。
- 隣接関係が対称であることを示す。
- 自己ループがないことを示す。

### 次数

- 任意の頂点 `i` について、近傍が `{i-1, i+1, i+4}` であることを示す。
- これら三つが互いに異なることを示す。
- よって次数が3であることを示す。

### 直径

- `0` からの距離層を列挙する。
- 回転 `r(i)=i+1` が自己同型であることを使い、任意の頂点を `0` に移せることを示す。
- 任意の頂点対の距離が2以下であることを示す。
- 非隣接な頂点対、例えば `0` と `2` が存在することを示す。
- よって直径が2であることを示す。

### 自己同型群

- `r(i)=i+1` が自己同型であることを示す。
- `s(i)=-i` が自己同型であることを示す。
- `r` と `s` が二面体群の関係式を満たすことを示す。
- `Aut(M8)` が頂点推移的であることを示す。
- `0` の安定化群の位数が2であることを示す。
- よって `|Aut(M8)|=16` を示す。
- 最終的に `Aut(M8)` が位数16の二面体群と同型であることを示す。

## 未確認事項

- 上記の自己同型群の議論は自然言語としては標準的だが、Lean 化するには「辺 `0--4` が近傍内で構造的に特徴づけられる」という部分をより厳密な補題に分ける必要がある。
- 辺推移でないことの証明も、4-cycle の本数を数える補題として切り出すとよい。
